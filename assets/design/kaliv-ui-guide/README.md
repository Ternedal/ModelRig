Denne pakke omsætter det godkendte Kaliv UI-koncept til en konkret implementeringsguide.

## Start her
- `Kaliv_UI_Design_Guide.pdf` - læsevenlig, endelig designguide
- `Kaliv_UI_Design_Guide.docx` - redigerbar version
- `Kaliv_UI_Target_Mockup.png` - visuelt mål for dark og light mode
- `Kaliv_UI_Annotated_Mockup.png` - nummereret layoutreference
