# Kaliv — Thinking Animation

En sømløs tænke-/loading-animation baseret på den godkendte Kaliv-ankh.

## Primær fil
- `GIF/kaliv_thinking_transparent_128.gif`

## Størrelser
- 64×64: inline emoji og små statusindikatorer
- 128×128: anbefalet til chat og thinking state
- 256×256: større panels og preview

## Varianter
- Transparent GIF
- Dark-mode GIF
- Light-mode GIF
- Transparent lossless animated WebP
- Transparent APNG

## Animation
- Ankh'en står stille og bevarer brandets identitet
- 12 bronze/guld-partikler bevæger sig som en loading-ring
- Diskret breathing og metallisk shimmer
- 32 frames
- 40 ms pr. frame
- 1.28 sekunders sømløst loop

GIF er medtaget som ønsket. Til Android anbefales normalt den animerede WebP,
fordi den har bedre transparens og ofte mindre filstørrelse.
