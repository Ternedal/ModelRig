# Claude Handoff — Kaliv Brand Package

## Use this package as source of truth

The approved visual source is the branded/burned ankh symbol, not the previous minimalist icons.

## Android adaptive icon

Use:

- `01_ANDROID_ADAPTIVE_ICON/ic_launcher_foreground.png`
- `01_ANDROID_ADAPTIVE_ICON/ic_launcher_background.png`
- `01_ANDROID_ADAPTIVE_ICON/ic_launcher_monochrome.png`

Requirements satisfied:

- foreground: transparent PNG, 1024×1024
- motif max approx. 40% of canvas
- background is separate, not baked into foreground
- monochrome themed icon included

## Product / UI direction

Kaliv is a chatbot / chatAI assistant with a dark, premium, tactile brand:

- local intelligence
- privacy-first
- dark chat UI
- bronze/ember accents
- ankh as identity mark
- charred materiality / branded seal feel

## Do not use

- old Alva name
- previous flat/wellness-looking icon directions
- foregrounds with background baked in
- oversized adaptive icon foregrounds
